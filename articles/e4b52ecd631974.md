## MoonbitでAtCoderの問題を解くには

現時点（2026年2月）では、AtCoderはMoonbitに対応していません。そのため、MoonbitをJavaScriptにコンパイルして提出してみることにします。

Mac / Homebrew環境を想定しているので、お使いの環境に合わせて適宜読み替えてください。

## ツールのセットアップ

ビルドやテスト、提出までの手順が煩雑なので、`just`と`online-judge-tools`を使うことにします。

```bash
# justのインストール
brew install just

# online-judge-toolsのインストール
pipx install "git+https://github.com/online-judge-tools/oj.git"
```

主に以下の3つのコマンドを、順に使用します。

- `just prepare <dir>`
  - テンプレートをコピーして、回答用ディレクトリを作成する
- `just download <URL>`
  - 問題のサンプルテストケースをダウンロードする
- `just test`
  - サンプルテストケースを実行し、正解したら提出用コードをクリップボードに貼り付ける
  - 提出用コードは、生成したJavaScriptをterserで圧縮し、Moonbitのコードを先頭にコメントとして挿入したもの

::::details justfile
:::message

```justfile
set quiet

alias p := prepare
alias d := download
alias t := test

[private]
resolve-dir:
  cat .atcoder-target

[private]
mbt-comment file:
  ( echo "/*"; cat "{{file}}"; echo "*/" )

prepare dir:
  cp -R template "{{dir}}"
  printf "%s\n" "{{dir}}" > .atcoder-target
  echo "created {{dir}} and set as current target"

use dir:
  printf "%s\n" "{{dir}}" > .atcoder-target
  echo "current target: {{dir}}"

current:
  cat .atcoder-target || echo "(not set)"

download url:
  cd "$(just resolve-dir)" && oj download "{{url}}"

build:
  moon build --target js --release "$(just resolve-dir)"

test:
  dir="$(just resolve-dir)" && just build && cd "$dir" && oj test -c "node ../_build/js/release/build/$dir/$dir.js"
  just copy

[script]
copy:
  dir="$(just resolve-dir)"
  (
    just mbt-comment "$dir/main.mbt"
    terser "_build/js/release/build/$dir/$dir.js" \
      --compress toplevel=true,passes=3 \
      --mangle toplevel=true
  ) | pbcopy
  echo "copied minified JS with source block comment header for $dir"
```

## [ABC081B - Shift only](https://atcoder.jp/contests/abs/tasks/abc081_b)

問題概要: N個の整数`A_1`, ... `A_N`がある。すべての整数が偶数のとき、全てを2で割ったものに置き換える操作が行える。最大で何回操作が行えるか求めてください。

### 提出

[提出 #73560496 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73560496)

```mbt
///|
fn main {
  let input = read_stdin()
  guard input.trim().split("\n").to_array() is [_n, a]
  let a = a.split(" ").map(to_int).to_array()

  let mut ans = 30
  for num in a {
    let mut count = 0
    let mut num = num
    while num % 2 == 0 {
      num /= 2
      count += 1
    }
    ans = Int::min(ans, count)
  }
  println(ans)
}
```

別解

```mbt
///|
fn div_count(num : Int) -> Int {
  let mut num = num
  let mut count = 0
  while num % 2 == 0 {
    num /= 2
    count += 1
  }
  count
}

///|
fn main {
  let input = read_stdin()
  guard input.trim().split("\n").to_array() is [_n, a]
  let a = a.split(" ").map(to_int)
  let ans = a.map(div_count).fold(init=30, Int::min)
  println(ans)
}
```

## [ABC083B - Some Sums](https://atcoder.jp/contests/abs/tasks/abc083_b)

問題文: 1以上N以下の整数のうち、10進法での各桁の和がA以上B 以下であるものの総和を求めてください。

### 解答

[提出 #73561637 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73561637)

```mbt
///|
fn digit_sum(x : Int) -> Int {
  // 各桁の和を求める
  let mut x = x
  let mut sum = 0
  while x > 0 {
    sum += x % 10
    x /= 10
  }
  sum
}

///|
test "digit_sum" {
  assert_eq(digit_sum(23), 5)
  assert_eq(digit_sum(3141), 9)
}

///|
fn main {
  let input = read_stdin()
  guard input.trim().split(" ").map(to_int).to_array() is [n, a, b]

  let mut ans = 0
  for i in 1..<=n {
    let sum = digit_sum(i)
    if a <= sum && sum <= b {
      ans += i
    }
  }
  println(ans)
}
```

## [ABC088B - Card Game for Two](https://atcoder.jp/contests/abs/tasks/abc088_b)

### 解答

降順ソート。sort + rev、sort_by、sort_by_keyなどが使える。

[提出 #73562167 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73562167)

///|
fn main {
  let input = read_stdin()
  guard input.trim().split("\n").to_array() is [_n, a]
  let a = a.split(" ").map(to_int).to_array()
  a.sort()
  let a = a.rev()

  let mut ans = 0
  for i, num in a {
    ans += if i % 2 == 0 { num } else { -num }
  }
  println(ans)
}

## [ABC085C - Otoshidama](https://atcoder.jp/contests/abs/tasks/abc085_c)

問題概要: 10000円札、5000円札、1000円札が計N枚あります。合計Y円になる組み合わせを一つ出力してください。

### 解答

N <= 2000なので、10000円札の枚数と5000円札の枚数をループして、O(N^2)で解きます。

[提出 #73579778 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73579778)

```mbt
///|
fn solve(n : Int, y : Int) -> (Int, Int, Int) {
  for i = 0; i <= n; i = i + 1 {
    for j = 0; i + j <= n; j = j + 1 {
      let k = n - (i + j)
      let sum = 10000 * i + 5000 * j + 1000 * k
      if sum == y {
        return (i, j, k)
      }
    }
  }
  return (-1, -1, -1)
}

///|
fn main {
  guard read_stdin().trim().split(" ").map(to_int).to_array() is [n, y]
  let ans = solve(n, y)
  println("\{ans.0} \{ans.1} \{ans.2}")
}
```

## [ABC049C - 白昼夢](https://atcoder.jp/contests/abs/tasks/arc065_a)

問題概要: 文字列Sが与えられます。`dream`、`dreamer`、`erase`、`eraser`の4つの文字列を組み合わせてSを作れるかどうか判定してください。

### 解答

[提出 #73579962 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73579962)

```mbt
///|
fn main {
  let s = read_stdin().trim()
  let n = s.length()
  let ts = ["dream", "dreamer", "erase", "eraser"]

  let dp = Array::make(n + 1, false)
  dp[0] = true
  for i in 0..<n {
    if dp[i] {
      for t in ts {
        // sのi文字目以降が、tから始まっていればOK
        if i + t.length() > n {
          continue
        }
        let slice = s[i:i + t.length()] catch { _ => "" }
        if slice == t {
          dp[i + t.length()] = true
        }
      }
    }
  }
  println(if dp[n] { "YES" } else { "NO" })
}
```

`StringView::has_prefix`を使うと簡潔に書けます。

```mbt
///|
fn main {
  let s = read_stdin().trim()
  let n = s.length()
  let ts = ["dream", "dreamer", "erase", "eraser"]

  let dp = Array::make(n + 1, false)
  dp[0] = true
  for i in 0..<n {
    if !dp[i] {
      continue
    }
    for t in ts {
      let ok = (try! s[i:]).has_prefix(t)
      if ok {
        dp[i + t.length()] = true
      }
    }
  }
  println(if dp[n] { "YES" } else { "NO" })
}
```
