## Moonbitとは

Rustと関数型言語を足して2で割った感じの言語だと認識しています。Wasm、JavaScript、Nativeなどにコンパイルできます。

https://www.moonbitlang.com/

Rustと関数型言語と書きましたが、私はどちらもほとんど使ったことがありません。なので、堅牢なTypeScriptだと思って雰囲気で書いていますが、意外となんとかなります。パターンマッチやパイプラインが便利です。

## どのようにMoonbitを学ぶか

ブラウザでMoonbitを体験できるTourがあるので、最初はこれをやるのがおすすめでしょうか。[MoonBit Language Tour](https://tour.moonbitlang.com/)

実際に使うとなると、モジュールシステム、エラーハンドリング、テストなどの情報が必要になるので、適宜ドキュメントを読んでいました。[MoonBit Language — MoonBit v0.7.1 documentation](https://docs.moonbitlang.com/en/latest/language/index.html)

AIの力を借りる場合は、`~/.moon/lib/core`に標準ライブラリがあることを伝えると、色々書き方を教えてもらえると思います。なお、[AtCoderのアルゴリズムコンテストでは生成AIの利用が禁止されている](https://info.atcoder.jp/entry/llm-rules-ja)ため、コンテストに参加する場合は注意が必要です。

標準ライブラリに何があるかは、Moonbitのパッケージレジストリの[mooncakes.io](https://mooncakes.io/docs/moonbitlang/core)が確認しやすいです。`Set`、`Map`、`Queue`、`Stack`、`PriorityQueue`など、必要なものは一通り揃っていると思います。

あとは、`moon doc [Symbol]`コマンドでAPIが見れます。例えば、`moon doc String`、`moon doc Array`のように使います。インポートが必要なものは、`moon doc @set.Set`のように先頭に`@モジュール名`と付けます。

`justfile`で主に使うコマンドは以下の3つです。

- `just prepare <dir>` / `just p <dir>`
  - テンプレートをコピーして、回答用ディレクトリを作成する
- `just download <URL>` / `just d <URL>`
  - 問題のサンプルテストケースをダウンロードする
- `just test` / `just t`
  - サンプルテストケースを実行し、正解したら提出用コードをクリップボードに貼り付ける
  - 提出用コードは、生成したJavaScriptをterserで圧縮し、先頭にMoonbitのコードをコメントとして挿入したもの

::::details justfile

```justfile
set quiet

alias p := prepare
alias d := download
alias t := test

[private]
resolve-dir:
  cat .atcoder-target

[private]
mbt-comment file:
  ( echo "/*"; cat "{{file}}"; echo "*/" )

prepare dir:
  cp -R template "{{dir}}"
  printf "%s\n" "{{dir}}" > .atcoder-target
  echo "created {{dir}} and set as current target"

use dir:
  printf "%s\n" "{{dir}}" > .atcoder-target
  echo "current target: {{dir}}"

current:
  cat .atcoder-target || echo "(not set)"

download url:
  cd "$(just resolve-dir)" && oj download "{{url}}"

build:
  moon build --target js --release "$(just resolve-dir)"

test:
  dir="$(just resolve-dir)" && just build && cd "$dir" && oj test -c "node ../_build/js/release/build/$dir/$dir.js"
  just copy

[script]
copy:
  dir="$(just resolve-dir)"
  (
    just mbt-comment "$dir/main.mbt"
    terser "_build/js/release/build/$dir/$dir.js" \
      --compress toplevel=true,passes=3 \
      --mangle toplevel=true
  ) | pbcopy
  echo "copied minified JS with source block comment header for $dir"
```

## [ABC081B - Shift only](https://atcoder.jp/contests/abs/tasks/abc081_b)

問題概要: N個の整数`A_1`, ... `A_N`がある。すべての整数が偶数のとき、全てを2で割ったものに置き換える操作が行える。最大で何回操作が行えるか求めてください。

### 解答

2で割れる回数の最小値を求めればよいです。素直にループを書きました。

[提出 #73560496 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73560496)

```mbt
///|
fn main {
  let input = read_stdin()
  guard input.trim().split("\n").to_array() is [_n, a]
  let a = a.split(" ").map(to_int).to_array()

  let mut ans = 30
  for num in a {
    let mut count = 0
    let mut num = num
    while num % 2 == 0 {
      num /= 2
      count += 1
    }
    ans = Int::min(ans, count)
  }
  println(ans)
}
```

### 別解

`map`と`fold`で宣言的に書いてみました。

```mbt
///|
fn div_count(num : Int) -> Int {
  let mut num = num
  let mut count = 0
  while num % 2 == 0 {
    num /= 2
    count += 1
  }
  count
}

///|
fn main {
  let input = read_stdin()
  guard input.trim().split("\n").to_array() is [_n, a]
  let a = a.split(" ").map(to_int)
  let ans = a.map(div_count).fold(init=30, Int::min)
  println(ans)
}
```

## [ABC083B - Some Sums](https://atcoder.jp/contests/abs/tasks/abc083_b)

問題文: 1以上N以下の整数のうち、10進法での各桁の和がA以上B以下であるものの総和を求めてください。

### 解答

[提出 #73561637 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73561637)

```mbt
///|
fn digit_sum(x : Int) -> Int {
  // 各桁の和を求める
  let mut x = x
  let mut sum = 0
  while x > 0 {
    sum += x % 10
    x /= 10
  }
  sum
}

///|
test "digit_sum" {
  assert_eq(digit_sum(23), 5)
  assert_eq(digit_sum(3141), 9)
}

///|
fn main {
  let input = read_stdin()
  guard input.trim().split(" ").map(to_int).to_array() is [n, a, b]

  let mut ans = 0
  for i in 1..<=n {
    let sum = digit_sum(i)
    if a <= sum && sum <= b {
      ans += i
    }
  }
  println(ans)
}
```

## [ABC088B - Card Game for Two](https://atcoder.jp/contests/abs/tasks/abc088_b)

問題概要: N枚のカードがあり、i枚目のカードには数a_iが書かれています。AliceとBobが、大きい数字が書かれたカードを交互に取っていきます。得点差がいくつになるか求めてください。

### 解答

降順ソートをして順番に解いていきます。`moon doc Array`を見ると、`Array::sort`、`Array::sort_by`、`Array::sort_by_key`などの関数があるのがわかります。

```
$ moon doc Array | grep sort
  pub fn[T : Compare] Array::is_sorted(Self[T]) -> Bool
  pub fn[T : Compare] Array::sort(Self[T]) -> Unit
  pub fn[T] Array::sort_by(Self[T], (T, T) -> Int) -> Unit
  pub fn[T, K : Compare] Array::sort_by_key(Self[T], (T) -> K) -> Unit
```

reverseする関数は`Array::rev`がありました。ソートの関数は破壊的な一方、`Array::rev`は非破壊的なので注意します。

```mbt
///|
fn main {
  let input = read_stdin()
  guard input.trim().split("\n").to_array() is [_n, a]
  let a = a.split(" ").map(to_int).to_array()
  a.sort()
  let a = a.rev()

  let mut ans = 0
  for i, num in a {
    ans += if i % 2 == 0 { num } else { -num }
  }
  println(ans)
}
```

## [ABC085C - Otoshidama](https://atcoder.jp/contests/abs/tasks/abc085_c)

問題概要: 10000円札、5000円札、1000円札が計N枚あります。合計Y円になる組み合わせを一つ出力してください。

### 解答

N <= 2000なので、10000円札の枚数と5000円札の枚数をループして、O(N^2)で解きます。

```mbt
///|
fn solve(n : Int, y : Int) -> (Int, Int, Int) {
  for i = 0; i <= n; i = i + 1 {
    for j = 0; i + j <= n; j = j + 1 {
      let k = n - (i + j)
      let sum = 10000 * i + 5000 * j + 1000 * k
      if sum == y {
        return (i, j, k)
      }
    }
  }
  return (-1, -1, -1)
}

///|
fn main {
  guard read_stdin().trim().split(" ").map(to_int).to_array() is [n, y]
  let ans = solve(n, y)
  println("\{ans.0} \{ans.1} \{ans.2}")
}
```

## [ABC049C - 白昼夢](https://atcoder.jp/contests/abs/tasks/arc065_a)

問題概要: 文字列Sが与えられます。`dream`、`dreamer`、`erase`、`eraser`の4つの文字列を組み合わせてSを作れるかどうか判定してください。

### 解答

dp[i] := i番目の文字まで一致させることができるか としてDPをします。

遷移ができるのは、dp[i] = trueかつ、sのi番目からi + |t|番目の文字がt(= dream | dreamer | erase | eraser)に一致する場合です。

文字列のスライスは`s[start:end]`のように書きます。`String::sub`も同じ関数です。

範囲外参照のために`CreatingViewError`をスローするようになっているので、よしなにハンドリングします。

```mbt
///|
fn main {
  let s = read_stdin().trim()
  let n = s.length()
  let ts = ["dream", "dreamer", "erase", "eraser"]

  let dp = Array::make(n + 1, false)
  dp[0] = true
  for i in 0..<n {
    if dp[i] {
      for t in ts {
        // sのi文字目以降が、tから始まっていればOK
        if i + t.length() > n {
          continue
        }
        let slice = s[i:i + t.length()] catch { _ => "" }
        if slice == t {
          dp[i + t.length()] = true
        }
      }
    }
  }
  println(if dp[n] { "YES" } else { "NO" })
}
```

[提出 #73579962 - AtCoder Beginners Selection](https://atcoder.jp/contests/abs/submissions/73579962)

`StringView::has_prefix`を使うと簡潔に書けます。ここでは範囲外参照をしないことが明らかなので`try!`で握りつぶします。

```mbt
///|
fn main {
  let s = read_stdin().trim()
  let n = s.length()
  let ts = ["dream", "dreamer", "erase", "eraser"]

  let dp = Array::make(n + 1, false)
  dp[0] = true
  for i in 0..<n {
    if !dp[i] {
      continue
    }
    for t in ts {
      let ok = (try! s[i:]).has_prefix(t)
      if ok {
        dp[i + t.length()] = true
      }
    }
  }
  println(if dp[n] { "YES" } else { "NO" })
}
```
